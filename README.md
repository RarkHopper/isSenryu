# isSenryu()

指定された文章が川柳であるかを判定する関数群です  

形態素を解析し、それぞれの読みの長さを元に5:7:5のパターンでの区切りがあれば川柳と判定します

```
朝起きて今日も会社に遅刻する
(形態素とそれぞれの読み) -> アサ(2) / オキ(2) / テ(1) / キョウ(2) / モ(1) / カイシャ(3) / ニ(1) / チコク(3) / スル(2)
(読みを一つずつ足していったときの合計値) -> 2, 4, 5, 7, 8, 11, 12, 15, 17
```

となり、読みの合計値に `5`、`12(5 + 7)`、`17(5 + 7 + 5)` の区切れがそれぞれあるので川柳とします

## 実行

事前に以下のコマンドで仮想環境やライブラリのインストールを済ませてください

```bash
uv sync
```

mecabや辞書のインストールは、notebookに含まれているため上からセルを実行してください
